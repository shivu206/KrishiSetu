# KrishiSetu

AI-Powered Satellite-Based Crop Monitoring & Intelligent Irrigation Advisory System

## Overview

KrishiSetu is a full-stack web platform that leverages satellite imagery, computer vision, and artificial intelligence to assist farmers in crop monitoring and irrigation planning.

## Tech Stack

### Frontend
- React
- Vite
- Tailwind CSS

### Backend
- Node.js
- Express.js
- MongoDB

### AI Service
- Python
- OpenCV
- PyTorch
